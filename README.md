# Day 10 – JavaScript Deep Theory & Practical Implementation

This repository contains Day 10 learning and implementation tasks focused on advanced JavaScript concepts and practical project development.

---

##  Deep Theory
The `deep_theory.md` file includes important theoretical concepts such as:

- Putting JavaScript fundamentals together
- Architecting a pure JavaScript application
- Optimizing loops, rendering, and data flow
- LocalStorage and SessionStorage patterns

These concepts helped in understanding how to structure scalable and maintainable JavaScript applications.

---

##  Project 1: Persistent Todo App
Folder: `persistent-todo-app`

This project is a Todo application built using pure JavaScript with persistent storage support.

### Features
- Add new task
- Edit existing task
- Delete task
- Data stored using `localStorage`
- Tasks remain after page refresh

### Files
- `index.html`
- `style.css`
- `script.js`
